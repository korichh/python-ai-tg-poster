# Usage

## Python

### Create .venv

```
python -m venv .venv
```

### Activate .venv

```
source .venv/bin/activate (.venv/Scripts/activate)
```

### Install all packages

```
pip install -r requirements.txt
```

### Run server

```
python src/index.py
```

### Save all packages

```
pip freeze > requirements.txt
```

### Deactivate .venv

```
deactivate
```
